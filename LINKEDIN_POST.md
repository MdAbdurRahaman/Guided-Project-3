🚀 **Built an Industrial End-to-End MLOps Wine Quality AI Pipeline with MLflow & Multi-Cloud Deployment!**

In real-world data science, writing code in a Jupyter Notebook is only 20% of the job. Moving a machine learning model into production requires robust schema validation, hyperparameter experiment tracking, modular software architecture, and automated CI/CD pipelines.

To master production MLOps engineering, I built a complete **Wine Quality Prediction AI Engine** using Python, Scikit-Learn (ElasticNet), MLflow, Flask, Streamlit, Docker, and GitHub Actions! 🍷🎯

---

### 💡 Key Technical Highlights & Pipeline Stages:
1️⃣ **Data Ingestion & Fail-Fast Schema Guardrails**: Automatically ingests raw dataset and validates features and data types against custom `schema.yaml` specifications to prevent silent downstream pipeline failures.
2️⃣ **Modular Package Architecture (`src/mlProject/`)**: Implemented clean stage-by-stage pipeline components for data transformation, train/test splitting, and model fitting.
3️⃣ **MLflow Experiment Tracking & Governance**: Tracks hyperparameters (`alpha`, `l1_ratio`), metrics (**RMSE: 0.6543, MAE: 0.5186, R²: 0.3379**), and registers trained model artifacts into the MLflow Model Registry.
4️⃣ **Dual Interactive Web Dashboards**: Developed both a Flask web application (with dark glassmorphism UI) and an interactive **Streamlit Dashboard** for real-time wine quality inference.
5️⃣ **DevOps & Cloud Infrastructure**: Containerized with **Docker** for AWS EC2 deployment via GitHub Actions CI/CD, and deployed live on **Streamlit Community Cloud**.

---

### 🔑 Key Learnings & Takeaways:
- **Modular Code over Monolithic Notebooks**: Separating configuration managers, dataclasses, and component modules makes ML systems scalable, maintainable, and enterprise-ready.
- **Automated Experiment Governance**: MLflow eliminates manual tracking spreadsheet chaos by providing versioned metric logging and model governance.
- **Business Interpretability**: Translating chemical inputs (acidity, pH, alcohol content) into actionable quality scores turns data into real-world manufacturing strategy.
